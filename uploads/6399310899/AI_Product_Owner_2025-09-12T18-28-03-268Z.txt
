 Visión de producto:
Una calculadora básica sencilla y intuitiva para realizar operaciones matemáticas elementales con facilidad.

Historias de usuario:
1. Como usuario, quiero poder sumar dos números para realizar la operación correctamente.
2. Como usuario, quiero poder restar dos números para obtener el resultado correcto.
3. Como usuario, quiero poder multiplicar dos números para obtener el producto correcto.
4. Como usuario, quiero poder dividir dos números y obtener el resultado correcto con la opción de manejar divisiones con el cociente entero o decimal.

Criterios de aceptación:
1. La calculadora debe mostrar el resultado correcto para cada operación matemática (suma, resta, multiplicación y división).
2. El usuario debe tener la capacidad de ingresar números manualmente o seleccionarlos de una lista predeterminada.
3. La calculadora debe manejar decimales correctamente en todas las operaciones, incluyendo divisiones.
4. El diseño y presentación del resultado deben ser fáciles de entender y acceder, con la opción de elección de un tema visual para personalizar la interfaz.

Requisitos no funcionales:
1. La calculadora debe poder ejecutarse en dispositivos móviles y computadoras.
2. Debe tener alta velocidad y rendimiento en las operaciones matemáticas para una experiencia rápida y fluida para el usuario.
3. Debe ser seguro y proteger la privacidad de los datos ingresados por el usuario.
4. Debe ser accesible para personas con discapacidades visuales o motrices, incluyendo opciones para lector de pantalla y adaptadores de teclado.
