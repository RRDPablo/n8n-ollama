 Visión de producto
Una calculadora sencilla que proporcione operaciones matemáticas elementales para la realización de cálculos rápidos y fácilmente entendibles por usuarios de nivel básico.

Historias de usuario
1. Como usuario, quiero poder sumar dos números para hacer operaciones matemáticas básicas.
2. Como usuario, quiero poder restar dos números para resolver problemas y calcular diferencias.
3. Como usuario, quiero poder multiplicar dos números para encontrar el producto de dos cantidades.
4. Como usuario, quiero poder dividir dos números para obtener la fracción entre ellos.

Criterios de aceptación
- La calculadora debe mostrar los resultados de las operaciones matemáticas exactamente y sin errores.
- El usuario debe poder ingresar valores numéricos correctamente en el dispositivo utilizando teclado o pantalla táctil.
- En caso de que el usuario introduzca cifras incorrectas, la calculadora debe mostrar un mensaje de error clarificando cómo corregirlo.
- La calculadora debe poder guardar los últimos resultados para su uso posterior.

Requisitos no funcionales
- La calculadora debe ser rápida y reactiva en la ejecución de operaciones, asegurando un tiempo de respuesta menor a 1 segundo.
- Debe estar disponible en varios idiomas para facilitar su uso internacional.
- La calculadora debe tener una interfaz intuitiva y fácil de usar, adaptándose a dispositivos móviles y escritorio.
- Debe cumplir con estándares de seguridad y privacidad para proteger los datos del usuario.
