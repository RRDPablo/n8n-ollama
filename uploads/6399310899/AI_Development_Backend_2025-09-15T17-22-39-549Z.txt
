 Desarrollo del lógica para realizar las cuatro operaciones básicas matemáticas en la aplicación:

1. Suma
```python
def sum(num1, num2):
    return num1 + num2
```

2. Resta
```python
def rest(num1, num2):
    return num1 - num2
```

3. Multiplicación
```python
def mult(num1, num2):
    return num1 * num2
```

4. División
```python
def div(num1, num2):
    if num2 != 0:
        return num1 / num2
    else:
        raise ValueError("Division by zero is not allowed.")
```

En este código se han implementado las cuatro operaciones básicas matemáticas (suma, resta, multiplicación y división) utilizando Python.
