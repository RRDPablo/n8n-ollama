 Visión de producto: Aplicación móvil intuitiva y eficiente para realizar operaciones básicas matemáticas (suma, resta, multiplicación y división).

Historias de usuario:
1. Historia de usuario 1: Como usuario final quiero tener la opción de agregar números y seleccionar una operación aritmética para obtener el resultado correcto de manera instantánea.
   - frontendTasks: Diseño y desarrollo de la interfaz de usuario que permita agregar números e interactuar con los botones correspondientes a las operaciones aritméticas.
   - backendTasks: Desarrollo del lógica para realizar las cuatro operaciones básicas matemáticas y mostrar el resultado en la interfaz de usuario correspondiente.
2. Historia de usuario 2: Como desarrollador quiero crear una aplicación que cumpla con estándares de seguridad y privacidad requeridos para aplicaciones financieras o comerciales.
   - backendTasks: Integración de medidas de seguridad y privacidad en la lógica del backend de la calculadora, incluyendo protección contra inyecciones SQL y cifrado de datos sensibles.

Criterios de aceptación:
- La calculadora debe ser capaz de realizar correctamente las cuatro operaciones básicas matemáticas.
- El usuario debe poder ingresar los números y seleccionar la operación aritmética utilizando una interfaz intuitiva y amigable al usuario.
- El resultado final de cada operación debe mostrarse en pantalla inmediatamente después de presionar el botón correspondiente.
- La aplicación debe cumplir con los estándares de seguridad y privacidad requeridos para aplicaciones financieras o comerciales.

Requisitos no funcionales:
- Compatibilidad con dispositivos móviles Android e iOS:
   - frontendTasks: Diseño del código compatible con dispositivos Android e iOS en la interfaz de usuario y en la lógica de la aplicación.
   - backendTasks: Desarrollo de la API y lógica del backend compatible con las diferentes plataformas (Android e iOS).
- Diseño intuitivo:
   - frontendTasks: Implementación de un diseño intuitivo y fácil de usar, con una interfaz gráfica limpia y ordenada.
- Optimización del rendimiento:
   - frontendTasks: Implementación de optimizaciones para ofrecer un rendimiento ágil y eficiente, permitiendo al usuario realizar operaciones en tiempo real sin retrasos o errores.
   - backendTasks: Implementación de medidas de optimización para garantizar un rendimiento óptimo en el procesamiento de datos.
