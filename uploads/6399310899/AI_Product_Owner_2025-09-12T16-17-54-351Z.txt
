 Visión de producto:
Una calculadora básica simple y intuitiva para realizar operaciones matemáticas básicas rápidamente y fácilmente. La aplicación servirá como herramienta útil en la vida diaria de los usuarios que requieran hacer cálculos simples y precisos.

Historias de usuario:
1. Como usuario, quiero poder sumar dos números para obtener el resultado correcto.
2. Como usuario, quiero poder restar dos números para obtener el resultado correcto.
3. Como usuario, quiero poder multiplicar dos números para obtener el resultado correcto.
4. Como usuario, quiero poder dividir dos números para obtener el resultado correcto.
5. Como usuario, quiero poder acceder a la calculadora básica desde una interfaz intuitiva y fácil de usar.
6. Como usuario, quiero que la calculadora me permita verificar mi cálculo con un botón para comprobar el resultado.

Criterios de aceptación:
1. Las operaciones matemáticas sean precisas y correctas.
2. La interfaz de usuario sea intuitiva y fácil de usar.
3. El botón para comprobar el resultado funcione correctamente.
4. La calculadora sea capaz de manejar valores numéricos de forma precisa.

Requisitos no funcionales:
1. La aplicación debe ser compatible con múltiples plataformas (Android, iOS y web).
2. Debe soportar el idioma en uso local para un uso fácil por los usuarios de todo el mundo.
3. Debe ser optimizada para una rápida respuesta y rendimiento.
4. La aplicación debe estar disponible en línea sin costo adicional para los usuarios.
