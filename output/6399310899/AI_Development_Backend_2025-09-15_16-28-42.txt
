 Para cada una de las tareas especificadas, voy a proporcionar un ejemplo en Python utilizando Flask como microservicio.

**1. Implementación del servicio que calcule la suma.**
```python
from flask import Flask, jsonify
app = Flask(__name__)

@app.route('/sum', methods=['POST'])
def calculadora_sum():
    numeros = [float(num) for num in request.get_json().values()]
    resultado = sum(numeros)
    return jsonify({'result': resultado})
```
**2. Implementación del servicio que calcule la resta.**
```python
@app.route('/restar', methods=['POST'])
def calculadora_resta():
    numeros = [float(num) for num in request.get_json().values()]
    resultado = numeros[0] - numeros[1]
    return jsonify({'result': resultado})
```
**3. Implementación del servicio que calcule la multiplicación.**
```python
@app.route('/multiplicar', methods=['POST'])
def calculadora_multiplicacion():
    numeros = [float(num) for num in request.get_json().values()]
    resultado = 1
    for numero in numeros:
        resultado *= numero
    return jsonify({'result': resultado})
```
