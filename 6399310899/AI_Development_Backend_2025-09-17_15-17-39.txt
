 Como desarrollador backend, mi tarea consiste en programar el código correspondiente a las siguientes tareas:

**Sprint 1 - backendTasks:**
a. Codificación de lógica para las operaciones aritméticas básicas (PHP)
   - implementar funciones para realizar sumas, restas, multiplicaciones y divisiones
b. Validación de datos y seguridad en el procesamiento de operaciones (PHP)
   - agregar filtrado y escapado de datos para evitar inyección SQL y evitar vulnerabilidades en las funciones PHP

**Sprint 2 - backendTasks:**
a. Codificación de lógica para almacenar el historial de cálculos recientes en la base de datos (MySQL)
   - agregar tabla para almacenar el historial de cálculos, campos: id, usuario_id, operación, resultado y fecha
b. Implementación de funciones para consultar y recuperar el historial de cálculos recientes (PHP)
   - crear función que consulta la base de datos y devuelve un array con los últimos 10 cálculos

**Sprint 3 - backendTasks:**
a. Codificación de lógica para almacenar y recuperar preferencias del usuario (MySQL)
   - agregar tabla para almacenar las preferencias, campos: id, usuario_id, nombre_preferencia y valor
b. Integración de las preferencias del usuario en la interfaz de usuario (HTML/CSS/JavaScript)
   - pasar preferencias desde backend a frontend para personalizar el estilo y funcionalidad de la interfaz

**Sprint 4 - backendTasks:**
a. Codificación de lógica para compartir resultados y operaciones con API de redes sociales y mensajería instantánea (PHP)
   - agregar función que envía resultado de la operación a las APIs de las redes sociales o mensajería instantánea
b. Integración de las funciones de compartición en la interfaz de usuario (HTML/CSS/JavaScript)
   - crear botones de compartición y configurar los parámetros de la API para enviar el resultado
