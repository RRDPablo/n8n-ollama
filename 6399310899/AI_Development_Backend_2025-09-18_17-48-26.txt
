1. Implementar una API REST que permita hacer las operaciones básicas aritméticas (sumar, restar, multiplicar y dividir).
- Código de ejemplo:
```javascript
const express = require('express');
const app = express();
app.use(express.json());

// Ruta para la operación suma
app.get('/sum/:num1/:num2', (req, res) => {
  const num1 = parseFloat(req.params.num1);
  const num2 = parseFloat(req.params.num2);
  const result = num1 + num2;
  res.json({result});
});

// ... Rutas para las operaciones resta, multiplicación y división

app.listen(3000, () => console.log('App running on port 3000'));
```
